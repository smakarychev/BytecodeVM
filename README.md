# Bytecode Virtual Machine

Bytecode VM for a toy dynamically typed language.

* Classes:
  * property and method inheritance
  * can call to `super` class
* Garbage collection
* Closures
* Strings (not utf-8) and arrays
* NaN-boxing (with a separate build configuration)

The implementation is based on the awesome book ['Crafting Interpreters'](https://craftinginterpreters.com/)  by Robert Nystrom.

## Examples

The examples are located in `examples` folder and cover all language features.

## Building

The project has no external dependencies.

The solution files are generated by premake, which has a Lua problem of being
unable to work with unicode paths, so the project directory **MUST NOT** have any
special unicode characters in its path.

- Run `build.bat`, this will generate .sln solution file
